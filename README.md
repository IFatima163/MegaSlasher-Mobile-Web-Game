# Mega-Slasher
A 2D mobile and web clicker game targeting the best score without loosing

## Features
- Starting Menu
- Music and sound option to adjust to preference
- Start, restart, and quit option
- First-person POV
- Bombs to act as game-over trigger
- Game Over Screen

## Tech Stack
- **Frontend:** Unity UI
- **Backend:** Unity, C#
